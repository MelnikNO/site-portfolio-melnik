# Telegram Bot for Herzen University (РГПУ им. Герцена)

# [Репозиторий в Github](https://github.com/MelnikNO/Telegrambot_HSPU/tree/main)

Этот телеграм-бот создан для помощи студентам и преподавателям РГПУ им. А.И. Герцена. Он предоставляет удобный доступ к информации о преподавателях университета, позволяя искать их по фамилии, кафедре или ученой степени.

## ✨ Основные функции

*   **🔍 Поиск по фамилии**: Быстрый поиск преподавателя по его фамилии.
*   **🏫 Поиск по кафедре**: Просмотр списка преподавателей, работающих на выбранной кафедре.
*   **🎓 Поиск по ученой степени**: Фильтрация преподавателей по наличию ученой степени (кандидат или доктор наук).
*   **📋 Удобное меню**: Взаимодействие с ботом осуществляется через интуитивно понятные кнопки.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/MelnikNO/Telegrambot_HSPU.git
cd Telegrambot_HSPU
```

### 2. Настройка виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# или
venv\Scripts\activate  # Для Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Создание файла `.env`

В корневой папке проекта создайте файл `.env` и добавьте в него ваш токен:

```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

### 5. Запуск бота

```bash
python bot.py
```

После запуска вы увидите сообщение в терминале: `"Бот запущен и начал опрос сервера Telegram..."`.

## 🛠️ Используемые технологии

*   [**Python 3.8+**](https://www.python.org/)
*   [**python-telegram-bot**](https://python-telegram-bot.org/) (библиотека для работы с Telegram API)
*   [**python-dotenv**](https://github.com/theskumar/python-dotenv) (для управления переменными окружения)

## 📁 Структура проекта

*   `bot.py`: Основной файл, содержащий логику запуска бота и регистрации обработчиков.
*   `commands.py`: Содержит функции-обработчики для основных команд (поиск, меню и т.д.).
*   `keyboards.py`: Определяет клавиатуры и кнопки для взаимодействия с пользователем.
*   `states.py`: Содержит константы для управления состояниями диалога (ConversationHandler).
*   `.env`: Файл с конфиденциальными переменными окружения (не должен загружаться в репозиторий).
